# 玩转通用智能体之Claude Code

## 第十八章 养直觉：从生手到熟手

我年轻时学骑自行车，起头别提多费劲——又要顾平衡，又要蹬脚踏，又要看路，又要扶把，手忙脚乱。后来骑得多了，这些动作就成了本能，不必想，自个儿就做出来了。

用 Claude Code 也是这样。起头你觉着处处都得费脑筋：不晓得怎么跟它说话，不晓得什么时候该使它、什么时候该自己来。可用久了，你会养出一种直觉。

### 一

什么是直觉？

就是你凭感觉就能拿主意，不必特意去想。瞧见一个问题，你直觉里就有了数：这事 Claude Code 办不办得了，该走哪条路办，大约要几个来回。

这直觉打哪儿来？靠练，靠攒，靠摔跟头。

### 二

头一条，你得多用。

别光等碰上难处了才想起它。平日的零碎事也尽管使它——格式化代码、生成文档、找文件、解释报错。用得多了，你就摸清它什么在行、什么不在行。这跟交朋友一样：处久了，你就晓得对方的脾性。

### 三

第二条，要常归置规律。

每回用完，匀点工夫想想：这回成色如何？哪儿还能改进？提示词够不够清楚？

把心得记下来，攒成自己的一套门道。譬如我就摸出几条：

- 牵连多个文件的活，最好先让它把计划列出来
- 绕的逻辑要一步步来，别想一口吃成胖子
- 出错时，把整段报错都给它，别只甩一句「报错了」

这些规律，就是你那点直觉的根基。

### 四

第三条，要看别人怎么用。

Claude Code 的社区很热闹，不少人会晒自己的用法、提示词的模板、整套的流程。瞧瞧别人怎么做，能给你不少点拨。

可别照单全搬。各人的活法不同，对人家合适的，未必对你合适。把别人的经验嚼一嚼，掺上自己的情形，化成自己的路数。

### 五

末了一条，要沉得住气。

直觉不是一天养出来的。你兴许要用上几个月，乃至更久，才真正使得顺手。

这当口你会撞上各样的窝心事：它把你的意思领岔了，改出来的代码用不成，花的工夫比预想的长。这都是常事，别灰心。

熬下去，慢慢就找着感觉了。到那时候，Claude Code 就像是你的一截延伸：你心里想什么，它就晓得该做什么。

这便是从生手到熟手的那条路。
